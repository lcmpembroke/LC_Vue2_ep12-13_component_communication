This code follows episodes 12 to 13 of Laracasts Learn Vue 2 Step by Step. It covers:

Example #1: Custom Events - so one component notify another about a particular action or event that just took place  
Example #2: Event Dispatcher - create your own (Vue instance already implements the necessary interface) to allow for communication between Vue components
